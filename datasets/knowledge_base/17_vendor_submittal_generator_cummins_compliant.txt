SECTION 1 — PROPOSED CONFIGURATION
================================================================================

1.1  Equipment Model
     Cummins QSK60G18 — 3,500 kW Prime Rating, 3,850 kW Standby Rating
     Manufactured at: Cummins Pune Engine Plant, Survey 21, Balewadi, Pune
     Country of Origin: India (engine assembled in India from globally sourced components)

1.2  Configuration
     Quantity: 4 units
     A-Bus: GEN-A-01 and GEN-A-02 (N+1 redundancy serving A-Bus load)
     B-Bus: GEN-B-01 and GEN-B-02 (N+1 redundancy serving B-Bus load)

     REQ-GENSET-001 compliance: N+1 redundancy confirmed. At full design IT load,
     one generator per bus is available as a hot-standby spare.
     Loss of any single generator will NOT cause loss of critical supply.

1.3  Overload Capability
     REQ-GENSET-002 compliance: Each QSK60G18 is rated for 110% overload for 1 hour
     in every 12-hour period at an ambient temperature up to 50°C.
     This is a standard Cummins product rating and is fully compliant.

================================================================================
SECTION 2 — ELECTRICAL PERFORMANCE
================================================================================

2.1  Ratings
     Prime Power Rating           : 3,500 kW / 4,375 kVA at 0.8 PF, 415V, 3-phase, 50 Hz
     Standby Power Rating         : 3,850 kW / 4,813 kVA at 0.8 PF, 415V, 3-phase, 50 Hz
     Output Voltage               : 415V ± 0.5%, 3-phase, 4-wire, 50 Hz ± 0.5%
     Voltage Regulation (steady)  : ±1% of nominal
     Frequency Regulation         : ±0.25% in isochronous mode (electronic governor)

2.2  Startup Performance
     Start-to-load time           : 10 seconds maximum (cold start, -15°C to +50°C ambient)
     Load acceptance (step)       : 100% rated load step accepted in single step
                                    Voltage dip: <15% of nominal, recovery within 2 seconds
                                    Frequency dip: <10% of nominal, recovery within 3 seconds

     REQ-GENSET-003 compliance:
     Remote start: YES — via DEIF AGC-4 controller with dry contact input
     Remote stop: YES — via DEIF AGC-4 controller
     Remote alarms: YES — all alarms available via Modbus TCP/IP (see Section 5)
     Status monitoring: YES — Modbus TCP/IP per CM-MODBUS-QSK60-001 register map

================================================================================
SECTION 3 — ENGINE AND ALTERNATOR
================================================================================

3.1  Engine
     Engine Make/Model     : Cummins QSK60 — 60.0L, 16-cylinder, V-type, 4-stroke diesel
     Cooling System        : Radiator-cooled (standalone radiator included in scope)
     Fuel System           : Electronic common rail injection, HPCR technology
     Air Intake            : Dual turbocharged, charge air cooled
     Lube Oil System       : Pressure-fed with full-flow filtration
     Engine Protection     : Electronic protection for low oil pressure, high temperature,
                             overspeed (112% rated speed), overcrank, and coolant temperature

3.2  Alternator
     Make/Model            : Stamford HCI634H (coupled to Cummins QSK60)
     Excitation System     : Brushless, self-exciting
     Insulation Class      : Class H (temperature rise limited to Class F)
     IP Rating             : IP23 minimum
     THD Voltage           : <5% at full non-linear load (per IEC 60034-1)
     Voltage Regulation    : ±0.5% (AVR controlled, no load to full load)

================================================================================
SECTION 4 — FUEL SYSTEM
================================================================================

4.1  Fuel Type                : HSD (High Speed Diesel) to IS:1460-2000
4.2  Day Tank (per generator) : 2,000 litre capacity, mild steel with external coating
                                 Day tank included in scope of supply
4.3  Bulk Fuel Storage        : 200,000 litre capacity (supplied separately by civil contractor)
4.4  Fuel Consumption         : 890 litres/hour at 100% prime rating (declared, per BS 5514)
4.5  Runtime at Bulk Tank     : 224 hours continuous at 100% load (>9 days)
4.6  Fuel Transfer Pump       : Electric transfer pump (prime power from site UPS)

================================================================================
SECTION 5 — MONITORING AND COMMUNICATIONS
================================================================================

5.1  Local Control
     DEIF AGC-4 Automatic Generator Controller (included in scope)
     Functions: Auto-start, auto-stop, auto-transfer, parallel operation control,
                Modbus master/slave, SCADA interface, event log (last 500 events)
     Display: 7-inch colour touchscreen, full alarm annunciation

5.2  BMS Integration
     Protocol: Modbus TCP/IP (per REQ-GENSET-003)
     Hardware: Ethernet port on DEIF AGC-4 controller
     Register map reference: CM-MODBUS-QSK60-001 (submitted as SUB-09)

     Required BMS integration points (per specification):
     All the following points confirmed available via Modbus:
     - Generator run status (running/stopped)
     - Generator fault status (fault/healthy)
     - Output voltage (per phase)
     - Output current (per phase)
     - Output frequency
     - Load percentage
     - Engine oil pressure
     - Engine coolant temperature
     - Engine speed (RPM)
     - Battery charger status
     - Fuel level (day tank)
     - Remote start/stop command (write register)
     - Mains available status
     - Auto/Manual mode status

5.3  SNMP
     DEIF AGC-4 supports SNMP v2c and v3. MIB file provided in documentation package.

SECTION 7 — FACTORY ACCEPTANCE TEST PROGRAMME
================================================================================

7.1  Proposed FAT Date
     10-12 February 2026 at Cummins Pune Engine Plant, Balewadi

7.2  FAT Test Programme

     Day 1 (10 February):
     Test 1: Visual inspection and dimensional verification
     Test 2: Mechanical installation inspection (coupling, mounting, clearances)
     Test 3: Fuel system inspection and leak test
     Test 4: Cooling system inspection
     Test 5: Electrical system inspection (alternator, AVR, control panel wiring)
     Test 6: Insulation resistance test (alternator windings)
     Test 7: Functional energisation and no-load voltage/frequency check

     Day 2 (11 February):
     Test 8: Cold start from -5°C (cold soak test — facility available at Cummins)
     Test 9: Step load test — 0% to 100% in single step (load acceptance)
     Test 10: Load profile test — 25%, 50%, 75%, 100% load steps (15 minutes each)
     Test 11: Fuel consumption measurement at 100% prime rating
     Test 12: Overload test — 110% for 60 minutes (REQ-GENSET-002)
     Test 13: Modbus connectivity test — 100% of required BMS registers verified live

     Day 3 (12 February):
     Test 14: All alarm point simulation and verification (complete alarm list)
     Test 15: Remote start/stop via Modbus (simulated BMS command)
     Test 16: Engine protection trip tests (simulated low oil pressure, high temperature)
     Test 17: AGC controller function test — auto-start sequence
     Test 18: Documentation review — all certificates, manuals, and drawings verified

     NOTE: ALL tests listed above are factory tests, fully witnessed by attending parties.
     No tests are proposed for deferral to site commissioning.
